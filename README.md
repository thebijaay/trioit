# Trio IT Website

This is the official website for Trio IT, a technology and innovation hub based in Kathmandu, Nepal.

## Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript
*   **Version Control:** Git, GitHub

## Setup and Deployment

1.  Clone the repository: `git clone https://github.com/your-username/trio-it-website.git`
2.  Open `index.html` in your browser to view the website.

## Environment Variables

There are no environment variables required for this project.
